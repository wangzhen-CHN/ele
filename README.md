# vue2.0构建饿了么

## 主要依赖
- 基于vue@2.0
- 使用vue-cli搭建项目框架
- 使用vue-router进行页面路由切换
- 使用axios进行http请求获取数据
- mockjs模拟数据
- 使用less编写样式
- 使用[better-scroll](https://github.com/ustbhuangyi/better-scroll)进行联动滑动
